# Dieta Tracker 🥗

PWA single-file para acompanhar a dieta: dias no plano, refeições livres (🍕🍺🍬), fruta diária e evolução do peso com média móvel.

**App:** https://henriquetibucheski.github.io/dieta-tracker/

- Sem dependências: HTML + CSS + JS vanilla num arquivo só.
- Dados ficam no dispositivo (`localStorage`) — backup via export/import JSON na aba Dados.
- Funciona offline (service worker network-first).

Companheiro do [treino-tracker](https://github.com/henriquetibucheski/treino-tracker).
